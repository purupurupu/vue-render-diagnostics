## Commits

- 1 commit = 1 logical change
- Format: `<type>: <why-focused message>`
  - type: `feat`, `fix`, `refactor`, `chore`, `docs`, `test`
- Write messages in English
- Run `pnpm fmt` and `pnpm build` before committing

## Pull Requests

- Title and body in **English**
- Title: under 70 chars, format `<type>: <concise description>`
- Body: `## Summary` (bullet points) + `## Test plan` (checklist)
- Link related issues
